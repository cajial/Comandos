Conceptos powershell

1. Estructuras de Control y Menús

Son la base para crear scripts interactivos que se repiten hasta que el usuario decide salir.

Bucle Principal (Menú)

  • Usar do { ... } while ($op -ne "X") mantiene el menú activo hasta que el usuario elige la opción de salida.

Selección de Opciones

  • switch ($op) { ... }: Estructura más limpia que múltiples 'if' para manejar las opciones del menú.

Bucles de Datos

  • foreach ($i in $lista) { ... }: Utilizado para iterar sobre listas, como usuarios importados desde un archivo CSV.

2. Entrada/Salida y Visualización

Comandos básicos para interactuar con el usuario en la consola.

  • Write-Host "Mensaje": Escribe texto en la pantalla.
  • Read-Host "Pregunta": Captura datos introducidos por el usuario.
  • Clear-Host: Limpia la pantalla para mantener el menú ordenado.
  • Format-Table -AutoSize: Muestra resultados en una tabla ordenada y legible.
  • Write-Progress: Muestra una barra de progreso visual para tareas largas.

3. Gestión del Sistema y Procesos

Procesos

  • Get-Process: Lista los procesos activos.
  • Stop-Process -Name "X": Detiene un proceso específico.
  • Wait-Process: Pausa el script hasta que un proceso termine.
  • Start-Process: Inicia programas. Opciones útiles: -WindowStyle Maximized (pantalla completa) y -Verb RunAs (ejecutar como Administrador).

Otros Comandos de Sistema

  • Get-LocalUser: Muestra los usuarios locales de la máquina.
  • Get-History: Muestra el historial de comandos de la sesión actual.
  • shutdown.exe /s /t 60: Comando nativo para apagar el sistema (ej. en 60 segundos).

4. Active Directory (AD)

Scripts enfocados en la gestión de usuarios y dominios en servidor.

Información y Gestión

  • Get-ADForest / Get-ADDomain: Obtiene información básica del bosque y dominio.
  • New-ADUser: Crea nuevos usuarios en el directorio activo.
  • Remove-ADUser: Elimina usuarios.
  • Set-ADAccountPassword: Modifica la contraseña de un usuario existente.

Operaciones Masivas y Seguridad

  • Import-Csv: Lee archivos de datos para realizar altas o bajas masivas en bucle.
  • Read-Host -AsSecureString: Pide una contraseña de forma segura (ocultando caracteres).
  • ConvertTo-SecureString: Convierte contraseñas de texto plano a formato seguro (necesario para New-ADUser al importar desde CSV).

5. Redes

  • Test-Connection -Quiet: Realiza un ping. Con -Quiet devuelve True/False para usar en condicionales 'if'.
  • Get-NetAdapter: Lista adaptadores (se puede filtrar por Wi-Fi usando InterfaceType).
  • Get-NetIPAddress: Obtiene la configuración IP (usualmente filtrando por IPv4).
  • Get-NetRoute: Muestra la tabla de enrutamiento.
  • Disable-NetAdapterBinding: Deshabilita elementos específicos (protocolos/servicios) de un adaptador de red.

6. Manejo de Errores y Varios

  • try { ... } catch { ... }: Estructura fundamental para capturar errores y evitar que el script se detenga abruptamente, mostrando mensajes personalizados.
  • Test-Path: Verifica si un archivo existe antes de intentar usarlo (muy útil antes de un Import-Csv).
  • Start-Sleep: Pausa la ejecución del script durante un tiempo determinado (ej. milisegundos en barras de progreso).
